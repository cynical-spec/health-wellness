# Sehat Saathi — JioBharatIQ Health Assistant

A voice-first vernacular health and wellness assistant.
Speaks any Indian language. Gives Ayurvedic home remedies.

## Live Demo
https://cynical-spec.github.io/health-wellness/

## Setup
1. Open `index.html`
2. Replace `REPLACE_WITH_YOUR_KEY` with your OpenAI API key
3. Push to GitHub
4. Enable GitHub Pages — Settings > Pages > main branch
5. Share the URL

## What it does
User speaks a health complaint in any Indian language.
Assistant understands, asks one question, gives a warm
Ayurvedic home remedy in the same language.

## Note
API key is client-side for demo purposes only.
Rotate the key after sharing.
